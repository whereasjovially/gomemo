# memo

Memo Life For You

## Usage

```
NAME:
   memo - Memo Life For You

USAGE:
   memo [global options] command [command options] [arguments...]
   
VERSION:
   0.0.1
   
COMMANDS:
     new, n     create memo
     list, l    list memo
     edit, e    edit memo
     grep, g    grep memo
     config, c  configure
     help, h    Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```

## Installation

```
$ go get github.com/mattn/memo
```
